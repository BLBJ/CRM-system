# CRM-system
利用express+vue 构建一个权限管理系统，根据用户身份实现不同权限管理。
https://github.com/BLBJ/CRM-system/settings
### 系统介绍
用户通过注册登录进入网站系统，不同的身份对应不同的权限管理，管理员可以进行用户和权限管理，可根据不同用户身份改变UI内容，进行各自的身份操作.
